Welcome to XML example!

  • Request /xml/value page with PUT request, put "xmlTest" String parameter in a body of a request
  • Then request the page with GET request and get parameter back inside 'echo' tag of a XML response.

Example:

Request: <xmlTest> Mechanize framework test </xmlTest>

Response: <echo> Mechanize framework test </echo>
