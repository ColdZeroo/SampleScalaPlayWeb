File POST/PUT/GET example:

1. POST binary stream data to /files to create new randomly named file. Response has "302 Created" code and file url in "Location" header. Like this:

POST /files
Content-Type: application/pdf
with binary stream will return
302 Created
Location: /files/fGc82.pdf

2. PUT binary stream data to /files/:filename to create new file with given filename. Response has "200 OK" code and file url in "Location" header. Like this:

PUT /files/broken.pdf
with binary stream will return
200 OK
Location: /files/broken.pdf

3. Then get file created with GET request to /files/:filename
